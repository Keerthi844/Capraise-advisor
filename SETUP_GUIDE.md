## 🎯 Customization Guide

### Change Colors
Edit `tailwind.config.js` and modify the `primary` colors object

### Update Content
Open any page in `src/pages/` and update the text content

### Add Images
Place images in `src/assets/` (create if needed) and import in components

### Change Brand Name
Search and replace "Capraise" throughout the project

### Update Contact Info
Edit in `Footer.jsx` and `Contact.jsx`
